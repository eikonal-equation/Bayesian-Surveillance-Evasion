Bayesian Surveillance Evasion

Authors

  • Dongping Qi, Cornell University
  • David Bindel, Cornell University
  • Alexander Vladimirsky, Cornell University

Abstract

Most classical optimal control problems assume the environment to be either fully known or statistically known with a probability distribution. We consider a simple 2D continuous path planning problem with a completely unknown intensity of random termination. An Evader is trying to escape a domain while minimizing the cumulative detection risk to some static enemy Observers. Those Observers' surveillance intensity is not revealed to the Evader initially and has to be learned as the Evader conducts a repetitive planning task. We propose a new strategic planning algorithm which utilizes Gaussian process regression to model the unknown function and aims to deal with the exploration-exploitation dilemma. We illustrate our method on several examples with different numbers and locations of Observers.

Paper

D, Qi, D. Bindel and A. Vladimirsky, "Surveillance Evasion Through Bayesian Reinforcement Learning,"
